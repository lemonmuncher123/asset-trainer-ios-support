隱私政策

最後更新：2026年6月12日
簡而言之：您的組合數據儲存在您的設備上，開發者不收集任何數據。

1. 概述

Asset Trainer 是一款僅在本地執行的投資組合訓練模擬器。本政策說明應用程式處理哪些資料、哪些資料（如有）會離開您的裝置，以及您可以做出的選擇。簡而言之：您的組合資料儲存在您的裝置上，開發者不收集任何資料。

2. 開發者收集的內容：無

開發者不收集、不接收、不儲存、不出售、也不共享任何個人資料。本應用程式沒有帳戶系統、沒有云服務、沒有由開發者運營的伺服器，也沒有任何分析元件。您在應用程式中的任何操作都不會上報給開發者。

3. 您的資料僅儲存在您的裝置上

您錄入的一切 — 交易、資產、房產、債務、日誌與設定 — 都儲存在本裝置的本地資料庫中，以及您自己建立的備份或匯出檔案中（例如由您掌控的裝置備份，或您匯出並分享的檔案）。開發者無法訪問其中任何內容。

4. 傳送給行情資料提供方的代號

為獲取市場價格，應用程式僅將您所跟蹤資產的代號（例如“AAPL”或“BTC”）傳送給第三方行情資料提供方：Yahoo Finance、騰訊、東方財富、新浪、Coinbase、OKX，以及（僅在您配置後）Alpaca。您的交易、數量、餘額、備註與身份資訊絕不會被髮送。與任何網際網路請求一樣，您所連線的提供方在技術上會獲得您裝置的 IP 地址，並可能依據其自身的隱私政策記錄請求，開發者無法控制這些政策。價格請求由您的裝置直接發往提供方，不經過開發者。

5. 可選的第三方 API 金鑰

如果您選擇使用 Alpaca 作為報價來源，您輸入的 API 金鑰僅儲存在本裝置上，並且只會傳送給 Alpaca 用於驗證價格請求。您與 Alpaca 的帳戶及關係受 Alpaca 自身條款與隱私政策的約束。

6. 通知

經您許可，應用程式會安排本地通知 — 例如新的週期報告生成時。通知完全在本裝置上生成；不涉及任何推送服務或伺服器。您可以隨時在 iOS 設定中撤銷該許可權。

7. 無跟蹤、無分析、無廣告

本應用程式不含廣告、不含跟蹤器、不含第三方分析 SDK，不使用廣告識別符號，也不對您的裝置進行指紋識別。

8. 資料保留與刪除

您的資料會一直保留在本裝置上，直到您刪除為止。您可以刪除單個資產及其歷史（設定 ▸ 資料管理）、清除全部資料（“清除所有資料…”），或直接刪除應用程式 — 這會移除本地資料庫。由於開發者不持有您資料的任何副本，開發者也沒有可刪除或可交出的內容。停止市場資料同步或刪除應用程式，即視為撤回您對上述資料流的同意。

9. 兒童隱私

本應用程式並非面向兒童，且如上所述，不收集任何人的資料 — 包括兒童。

10. 本政策的變更

如本政策發生重大變更，更新後的版本將隨應用程式更新一同釋出，變更內容可在本文件中檢視。更新後繼續使用即表示接受修訂後的政策。

11. 聯絡方式

有關本政策的問題，可發送至開發者電郵 goatintegral@gmail.com。
